# Doc_genie
This project uses several Python libraries to support different functionalities. Gradio is used to create a web interface for user interaction. JSON5 and JSON Schema are used to process and validate JSON data. Pydantic helps with data validation, while python-dotenv manages environment variables.
For data handling and output, Pandas and OpenPyXL are used to analyze data and work with Excel files. Colorama and Tabulate improve output display. Development tools like PyTest, pytest-cov, Black, and Flake8 help in testing, formatting, and maintaining code quality.
